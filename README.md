# Office Lunch Roulette

**Live Demo (English):** https://nyungbin.github.io/office-lunch-roulette/en/

**Live Demo (Korean):** https://nyungbin.github.io/office-lunch-roulette/

**Repo:** https://github.com/nyungbin/office-lunch-roulette

A lightweight web app that solves the daily "where should we eat lunch?" debate. Enter your office or building name, pick a walking distance, and spin a roulette wheel to get a random restaurant pick from real nearby places.

> **Note:** Restaurant search currently only covers locations within South Korea. To run this locally, replace the Kakao API key in `index.html` with your own.
